## 一、标准文档流

### 1.1 隐藏元素

设置元素不可见的方式

- ​	display: none;   设置元素消失不占位
- ​	visibility: hidden;  设置元素消失占位
- ​	opcity: 0;  设置元素的不透明度为0，元素消失占位

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .container {
            height: 800px;
            background: pink;
        }
        .container div {
            width: 100px;
            height: 100px;
        }
        .box1 {
            background: green;
            /* 隐藏：不占位 */
            display: none;
        }
        .box2 {
            background: orange;
            /* 隐藏：占位 */
            visibility: hidden;
        }
        .box3 {
            background: skyblue;
            /* 透明隐藏：占位 */
            opacity: 0;
        }
        .box4 {
            background: gold;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="box1">box1</div>
        <div class="box2">box2</div>
        <div class="box3">box3</div>
        <div class="box4">box4</div>
    </div>
</body>
</html>
```



### 1.2 居中

- 单行文本在盒子里垂直居中。

- 多行文本在盒子垂直居中。盒子不给高度，用上下padding挤出来。

- 文本水平方向居中。

- 盒子在父盒子内水平居中。

- 盒子在父盒子内垂直居中。

    ​	不给父盒子设置高度，用子盒子的高度撑开。给父盒子设置同样的上下边距。

- 表格有在居中方面很有优势，我们也可以通过模拟表格来实现居中。

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title></title>
    <style>
        /* 单行为本居中 */
        .box1 {
            width: 200px;
            height: 200px;
            border: 1px solid #000;
            /* 水平方向 */
            text-align: center;
            /* 垂直方向 */
            /* 让行高与高度一致 */
            line-height: 200px;
        }
        .box2 {
            width: 200px;
            /* height: 200px; */
            border: 1px solid #000;
            margin-top: 20px;
            line-height: 30px;
            /* 水平方向 */
            text-align: center;
            /* 垂直方向 */
            /* 通过padding来设置高度 */
            padding: 90px 0;
        }
        .container {
            width: 300px;
            height: 300px;
            border: 1px solid #000;
            margin-top: 20px;
            /* 行内元素：设置容器元素 */
            /* 水平居中 */
            /* text-align: center; */
            /* 垂直居中 */
            /* line-height: 300px; */
            
            /* 让块元素居中，可以设置上下padding */
            /* padding: 200px 0; */

            /* 行内块元素 */
            /* 水平居中 */
            text-align: center;
            /* 垂直居中 */
            line-height: 300px;
        }
        .box3 {
            width: 100px;
            height: 100px;
            background: green;
            margin: 0 auto;
            /* 如果已知高度，设置margin-top，但是不要塌陷 */
            /* margin-top: 100px; */
            /* margin: 100px auto; */
            /* 如果父容器高度不确定，可以在父容器中，设置padding居中 */
        }
        img {
            width: 120px;
            height: 80px;
            /* 注意：垂直居中的时候，要更改图片的对其方式 */
            vertical-align: middle;
        }
        /* 模拟表格实现居中 */
        .ickt {
            width: 600px;
            height: 600px;
            border: 10px solid pink;
            /* 模拟表格 */
            display: table-cell;
            /* 水平居中 */
            text-align: center;
            /* 垂直居中 */
            vertical-align: middle;
        }
    </style>
</head>
<body>
    <!-- 通过表格居中 -->
    <!-- 借助于表格特性，我们可以将单行文本，多行文本，行内元素，块元素，行内块元素等，实现居中 -->
    <div class="ickt">
        hello
        <br>
        hello ickt;
        <span>span</span>
        <div>div</div>
        <img src="./img/7.jpg" alt="">
    </div>
    <!-- 表格 -->
    <table border="1" width="500" height="500" align="center">
        <tbody>
            <tr>
                <!-- th设置居中，会将字体加粗 -->
                <!-- 单行文本 -->
                <!-- <td align="center">
                    hello ickt!
                </td> -->
                <!-- 多行文本 -->
                <!-- <td align="center">
                    hello ickt!
                    <br>
                    hello ickt!
                </td> -->
                <!-- 行内元素 -->
                <!-- <td align="center">
                    <span>hello</span>
                </td> -->
                <!-- 块元素 -->
                <!-- <td align="center">
                    <div>hello ickt!</div>
                </td> -->
                <!-- 行内块元素 -->
                <td align="center">
                    <img src="./img/7.jpg" alt="">
                </td>
            </tr>
        </tbody>
    </table>
    <!-- 文本居中 -->
    <div class="box1">
        hello ickt!
    </div>
    <div class="box2">
        hello ickt!
        <br>
        hello ickt!
        hello ickt!
    </div>
    <!-- 盒子居中 -->
    <div class="container">
        <!-- 行内元素 -->
        <!-- <span>hello</span> -->
        <!-- <div class="box3">hello</div> -->
        <img src="./img/7.jpg" alt="">
    </div>
</body>
</html>
```

## 1.2 浮动特征

- 浮动可以使元素从左到右紧密排列；

- 浮动的元素具有块级元素的特点（可以设置宽高，margin、padding生效，但是没有换行符）；

- 浮动的元素宽度不再占满容器，而是变成了包裹内容的宽度。

- 浮动后的元素不存在上下外边距合并问题；

- 浮动的元素脱离了文档流，不再占有原来的位置；

- 一个元素设置浮动后，为了不影响后面元素渲染顺序，让渲染的结果具有一致性，要将其它同级

    的元素都要设置为浮动。

- 子元素设置浮动后，父元素如果没有设置高度，会呈现出塌陷的效果；

- 元素设置浮动后，会影响后面元素的布局，所以设置浮动后一定要及时清除；

- 清除浮动的属性  clear:left | right | both(常用);

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <style>
        .container {

            width: 800px;
            background-color: green;
        }
        .container div {
            /* width: 100px; */
            height: 100px;
```

### 1.3 浮动塌陷的问题

上面提到，在父元素不设置宽高的情况下，嵌套的子元素如果都设置了浮动，会呈现出父元素塌陷的

效果。这个问题，需要解决，否则有可能会影响页面的布局。解决方案：

1. ​	给父元素设置高度：这种方式在确定宽高的情况下，可以使用；但很多时候，容器都是由内容

    撑起来，所以根据内容的多少，不太好控制这个高度（改动较大）

    ```html
    .container {
        margin: 10px auto;
        /* width: 400px; */
        width: 800px;
        background-color: green;
        /* 第一种解决方案：父元素设置高度 */
        height: 200px;
    }
    ```

    

2. ​	外墙法：在父元素的后面，设置父元素的一个同级空元素（div），设置清除浮动；这种方式需

    要添加	过多的空元素，增加结构代码量；并且我们完全可以在后面那个同级父元素上直接清除

    浮动。

    ```html
    <head>
        <style>
            .outer {
                clear: both;
                height: 200px;
                background: yellowgreen;
            }
            /* 让外墙清除父元素的浮动 */
            /* .outer-wall {
                clear: both;
            } */
        </style>
    </head>
    <body>
        <div class="container">
            <div class="box1">1-1-1</div>
            <div class="box2">2-2-2</div>
            <div class="box3">3-3-3</div>
            <div class="box4">4-4-4</div>
            <div class="box5">5-5-5</div>
            <div class="box6">6-6-6</div>
            <div class="box4">4-4-4</div>
            <div class="box5">5-5-5</div>
            <div class="box6">6-6-6</div>
        </div>
        <!-- 外墙法 -->
        <div class="outer-wall"></div>
        <!-- 外面哟䘝元素 -->
        <div class="outer">
            外面的元素
        </div>
    </body>
    ```

    

3. ​	内墙法：在父元素内部后面，添加一个空的子元素（div），设置清除浮动；问题同外墙法类

    似，但是	解决了外墙法不能解决的父元素塌陷问题；相对于给父元素设置高度的不确定性，这

    种方式直接内容撑起父元素的高度。

    ```html
    <head>
        <style>
            /* 让内墙清除浮动 */
            .inner-wall {
                clear: both;
                display: block;
            }
        </style>
    </head>
    <body>
        <div class="container">
            <div class="box1">1-1-1</div>
            <div class="box2">2-2-2</div>
            <div class="box3">3-3-3</div>
            <div class="box4">4-4-4</div>
            <div class="box5">5-5-5</div>
            <div class="box6">6-6-6</div>
            <div class="box4">4-4-4</div>
            <div class="box5">5-5-5</div>
            <div class="box6">6-6-6</div>
            <div class="box6">6-6-6</div>
            <!-- 内墙法 -->
            <span class="inner-wall"></span>
        </div>
        <!-- 外面哟䘝元素 -->
        <div class="outer">
            外面的元素
        </div>
    </body>
    ```
